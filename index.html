Développeur web,
c'est quoi ? ↓

Les missions d’un développeur web

Le développeur web, c’est le responsable de la programmation. Il est en charge de la conception de sites, d’applications, de logiciels et de programmes informatiques « sur-mesure ».


  • il analyse d’abord leurs besoins
  • il dresse ensuite un cahier des charges précis
  • il écrit les lignes de code
  • enfin, il effectue des tests et d’adapter le logiciel selon les demandes du client.

↓

Après la création

Après la création du site et/ou de l’application, le développeur web a pour mission de le maintenir et de le faire évoluer quand cela est nécessaire.

Il veille aussi à la responsivité du site afin que celui-ci soit optimisé pour le mobile.


↓

En plus

↓

Le développeur web possède également une casquette de formateur. En effet, il peut être amené à rédiger des guides qui permettront aux utilisateurs de mieux comprendre les programmes créés. Ces explications servent aussi à former les personnes à éviter les erreurs ou à les régler plus facilement en cas de besoin.

L’apprentissage peut se faire à travers une formation individuelle ou d’un groupe d’employés.

Dev App Web

Les développeurs Web Apps que recherchent aujourd’hui les entreprises doivent être capables très souvent de travailler sur trois niveaux :

1) sur les technologies clients permettant aux utilisateurs de visualiser et d’interagir avec des interfaces de différentes natures (Laptop, smartphone, tablettes, …),

2) sur des traitements complexes côté serveur web

3) sur les interactions entre le serveur web et les systèmes de base de données.


↓

Dev App Mobile iOS/Android

Cette formation propose l'acquisition de compétences avancées sur les deux plateformes faisant aujourd'hui office de locomotive sur le marché des mobiles: IOS (iPhone/ iPad) et Google Android (Smartphone/ tablette).

A l'issue de cette formation, les stagiaires seront capables de créer des applications "typiques" sur iOS et Google Android mais aussi d'utiliser les fonctionnalités avancées des smartphones tels que les accéléromètres, les puces GPS, les interfaces multi-touch et les puces NFC.

Les notions essentielles d'ergonomie des applications mobiles seront également abordées durant cette formation.

Ambigramme

Un ambigramme est une représentation graphique d'un mot dont la forme permet une double lecture.

Le terme a été inventé par Douglas Hofstadter, en 1983.

Le plus souvent, les ambigrammes ont la forme de mots visuellement symétriques.
